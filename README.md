# pico-solar

## Wiring

| Pin  | Connection           |
| ---- | -------------------- |
| GP4  | SDA                  |
| GP5  | SCL                  |
| GP15 | DHT Data             |
| GP26 | Voltage Divider Vout |

### Voltage Divider

| Resistor | Value (ohm) |
| -------- | ----------- |
| R1       | 10,000      |
| R2       | 2,000       |
